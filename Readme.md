# 🛍️ Vrinda Store Sales Analysis 2022

## 📌 Project Overview

The **Vrinda Store Sales Analysis 2022** project is an Excel-based Data Analytics project focused on analyzing customer orders, sales performance, product categories, customer demographics, and sales channels.

The dataset contains more than **31,000+ sales records** from different e-commerce platforms such as Amazon, Myntra, Ajio, Flipkart, and others.

The main goal of this project is to transform raw business data into meaningful insights using:

* Data Cleaning
* Pivot Tables
* Pivot Charts
* KPI Metrics
* Dashboard Creation
* Business Analysis

This project helps understand:

✔ Customer purchasing behavior
✔ Monthly sales trends
✔ Gender-wise sales contribution
✔ Top-performing states
✔ Best sales channels
✔ Order delivery performance
✔ Age group analysis

# 📊 Sheet-by-Sheet Analysis

## 1️⃣ Sales vs Orders Analysis

### Sheet: `Sales vs order`

This sheet compares:

* Monthly Sales Revenue
* Monthly Order Count

### Analysis Performed

* Total sales aggregation
* Monthly order trend analysis
* Revenue comparison across months

### Insight

Sales remained consistent throughout the year with peak sales observed during festive seasons.

---

## 2️⃣ Men vs Women Sales Analysis

### Sheet: `men vs women`

This sheet analyzes gender-based contribution to revenue.

| Gender | Sales Contribution |
| ------ | ------------------ |
| Men    | 7.6M               |
| Women  | 13.5M              |

### Key Insight

Female customers generated significantly higher revenue compared to male customers.


## 5️⃣ Age Group Analysis

* Amazon is the highest-performing platform.
* Maharashtra is the top-performing state.
* Most orders are delivered successfully.
* Return and cancellation percentages are low.
